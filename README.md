## üìñ About The Project

The **BUCC Android App** is the official mobile application for the  
**BRAC University Computer Club (BUCC)**.

This application is designed to streamline club activities and provide
members with a centralized platform for communication and resource management.

It serves as a mobile companion to the BUCC ecosystem, allowing members to stay
updated with the latest events, announcements, and club activities directly
from their Android devices.
